## 项目介绍
大作业项目是爬去爬去网络数据制作小说排行榜，共有两部分，QidianSpider是数据爬去部分，主要爬小说的相关信息；qixqiView是数据展示部分，初步打算采用使用网页展示，采用flask网络框架


### QidianSpider
数据爬取，下去自己写个更好的


### qixqiView
数据展示
